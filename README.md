# GCSE Python

Free resources for learning Python for GCSE Computer Science (AQA 8525 / OCR J277).

Written for students, parents, and anyone starting out — all examples use **Thonny** and **Python 3**.

---

## Syllabus

The complete GCSE Python programming syllabus with explanations and code examples for every topic.

**[Read the syllabus](gcse_python_syllabus.md)**

Covers: data types, variables, casting, input/output, operators, selection, iteration, strings, lists, 2D arrays, dictionaries, functions, scope, file handling, random numbers, SQL, validation, searching, sorting, trace tables, errors, and testing.

## Getting Started

1. Download and install **[Thonny](https://thonny.org)** (free — comes with Python built in)
2. Read the [tips guide](learning_python_tips.md) for setup instructions
3. Work through the [syllabus](gcse_python_syllabus.md) topics in order
4. Do the exercises for each topic as you go
