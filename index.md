# GCSE Python

Free resources for learning Python for GCSE Computer Science (AQA 8525 / OCR J277).

Written for students, parents, and anyone starting out — all examples use **Thonny** and **Python 3**.

---

## What's here

- **[Syllabus]({% link gcse_python_syllabus.md %})** — every GCSE Python topic explained with code examples
- **[Tips for Parents and Kids]({% link learning_python_tips.md %})** — how to set up, practise, debug, and stay motivated
- **[Exercises]({% link exercises/index.md %})** — drills, challenges, and exam-style questions for each topic

---

## Getting Started

1. Download and install **[Thonny](https://thonny.org)** (free — comes with Python built in)
2. Read the [tips guide]({% link learning_python_tips.md %}) for setup instructions
3. Work through the [syllabus]({% link gcse_python_syllabus.md %}) topics in order
4. Do the [exercises]({% link exercises/index.md %}) for each topic as you go
